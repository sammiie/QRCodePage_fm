### What I learned

- how to center a div elements horizontally and vertically
  ```css
      div{
       positon: absolute;
       left: 50%;
       top: 50%;
       transform: translate(-50%, -50%);
        }
  ```
- I get to understand the different CSS positioning
  - `position:fixed;`
  - `position: absolute;`
  - `position: relative;`
- How to import google fonts and use them in my code
